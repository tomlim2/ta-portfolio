Matcap Painter

Three.js WebGPU
포토샵 왕복 없이 NPR 캐릭터용 Matcap 텍스처를 실시간으로 만들고, 3D 모델에서 바로 확인하는 도구
역할 Tool Design, Shader Development 기간 2026년 2월 (4일) 타입 개인 프로젝트
Three.js WebGPU (r172) Vanilla JavaScript Canvas 2D API
Shader Mode 조작 데모 — Color Ramp 조절 → 실시간 3D 프리뷰

동기

회사 NPR 캐릭터에 금속 질감과 머리카락 광택을 내기 위한 페이크 리플렉션 마스크용 Matcap 텍스처가 필요했다. 기존 워크플로우는 포토샵에서 수작업 → 엔진 임포트 → 확인 → 안 맞으면 다시 포토샵 반복. 결과를 보면서 실시간으로 조정할 수 없어 이터레이션이 느렸다.

해결

Shader Mode — 처음부터 만들 때

Blender Color Ramp 스타일 UI로 Toon Ramp, Specular 하이라이트, Outline을 실시간 조절. 밴드 수와 경계를 드래그하면 3D 프리뷰에 즉시 반영.

Paint Mode — 기존 것을 편집할 때

오픈소스 라이브러리(nidorx/matcaps)에서 큐레이션한 229개 프리셋을 기반으로, 8레이어 포토샵 스타일로 비파괴 편집. HSV, 콘트라스트, Lift를 조정하면서 실시간 결과 확인.

Paint Mode 레이어 편집 데모 — 8레이어 비파괴 편집 과정

편집 중인 Matcap 텍스처가 Sphere, Torus, 또는 GLB 모델에 실시간 적용. 포토샵↔엔진 왕복 없이 최종 결과를 바로 확인할 수 있다.

기술 상세

Per-pixel 노멀맵 렌더링
Canvas 2D에서 ray-sphere intersection으로 픽셀별 노멀을 계산하고, NdotL 라이팅을 적용. GPU 없이 CPU에서 실시간 Matcap 텍스처 생성.
Blender Color Ramp 재구현
Step(하드 밴드) / Linear(LERP) 모드. 밴드 수와 경계를 드래그로 조절하면 Toon Ramp 셰이딩 스타일이 즉시 변경. Blender 아티스트에게 익숙한 UI.
8레이어 컴포지팅
RGB → HSV per-pixel 변환, 더티 플래그 기반 캐싱으로 성능 확보. 레이어별 HSV/Contrast/Lift 비파괴 조정. Canvas → Three.js WebGPU CanvasTexture 실시간 파이프라인.
