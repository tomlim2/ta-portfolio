NPR Shader System

메타휴먼에서 애니메 스타일로 전환 — PBR 배경 위에 NPR 캐릭터를 분리 렌더링, 인하우스와 VRM 양쪽에 4종 룩 적용
Shader Development, Look Development, Material System 2023~2024 회사 프로젝트
Unreal Engine 5 Material Editor Post Process Stencil Buffer Custom Depth SDF Matcap

문제

프로젝트 초기 메타휴먼 기반 실사 렌더링에서 애니메 스타일로 전환이 결정됐다. PBR 배경은 유지하면서 캐릭터만 NPR로 렌더링해야 했고, 인하우스 캐릭터와 외부 VRM은 머티리얼 구조가 달라서 같은 방식으로는 양쪽 룩을 맞출 수 없었다.

해결

기존 플러그인과 셰이딩 기법을 조사해서 Cel Shading 기반 NPR 프로토타입을 만들었다. 이후 셰이더 전문 인력이 합류해 시스템을 확장했고, 해당 인력 퇴사 후 현재까지 기능 개발과 유지보수를 담당하고 있다.

아트팀이 Toon Ramp 텍스처만 교체하면 셰이딩 스타일이 바뀌는 구조로, 인하우스는 Custom Depth + Post Process, VRM은 Emissive + Inverted Hull로 나눠 4종 NPR 룩을 만들었다. 프로토타입에서 살아남은 핵심 노드는 SDF 얼굴 그림자와 Dot Product 기반 투톤 셰이딩이다.

SDF 얼굴 그림자

일반 NdotL로는 코, 눈두덩이 주변에 비직관적인 그림자가 생겨 얼굴이 깨진다. SDF(Signed Distance Field) 텍스처에 미리 구운 거리 정보로 그림자 경계를 제어하여, 어느 라이팅 각도에서든 얼굴 그림자가 자연스럽게 떨어지도록 만들었다.

📷 SDF 텍스처 원본: 얼굴 거리 맵 (흑백 그라데이션, 텍스처 에디터에서 캡처)
📷 얼굴 클로즈업 비교: 좌=NdotL 그림자(코/눈두덩이 깨짐) / 우=SDF 그림자(자연스러운 경계)
Material Node Graph
📷 머티리얼 에디터: SDF 텍스처 샘플 → Light Direction dot → 그림자 마스크 출력 노드 전체

툰 그림자 → 컬러 톤 전환

프로젝트 특성상 AI가 3D 모델과 카메라를 자동 제어하는 환경이라, 라이팅 방향이나 카메라 구도가 기술적으로는 맞아도 연출적으로 안 예쁜 경우가 자주 생겼다.

문제 — 밝고 행복한 씬에서 빛이 캐릭터 뒤에서 비추면, 투톤 그림자가 캐릭터를 "사악하고 음모를 꾸미는 듯한" 느낌으로 만들어버림. AI는 어디가 예쁘고 어디가 안 예쁜지 판단 불가.

해결 — 투톤 그림자를 끄고, 날씨/환경에 맞는 컬러 톤 변형 방식으로 전환. 라이팅 방향에 관계없이 캐릭터가 깨지지 않는 구조.

📷 뷰포트: 역광 씬에서 투톤 그림자 ON — 얼굴에 강한 명암 분리, 분위기 안 맞는 예시
📷 뷰포트: 동일 역광 씬에서 컬러 톤 방식 — 그림자 없이 안정적인 컬러 톤

베이스 머티리얼 3종

인하우스와 VRM은 머티리얼 구조가 근본적으로 다르고, VRM 안에서도 실사/비실사 파트가 혼재한다. 이를 해결하기 위해 세 종류의 베이스 머티리얼을 각각 구축했다.

1. 인하우스 툰

규약 없이 자유롭게 설계한 툰 전용 머티리얼. Custom Depth 버퍼로 캐릭터를 분리하고, Post Process에서 NPR 스타일로 다시 그린다. Inverted Hull로는 불가능한 내부 아웃라인(머리카락 갈래, 옷 주름 경계)까지 표현 가능.

📷 뷰포트: 인하우스 캐릭터 툰 렌더링 — 셀셰이딩 + 아웃라인 전신
📷 뷰포트: Custom Depth 버퍼 시각화 or 내부 아웃라인 클로즈업(머리카락 갈래, 옷 주름)
Material Node Graph — 인하우스 Post Process NPR
📷 머티리얼 에디터: Post Process Material — Custom Depth 마스킹 → SceneTexture Base Color → NPR 리라이팅 노드 전체

2. VRM 툰

MToon 규약에 정해진 플래그, 파라미터, 텍스처를 기반으로 — 베이스컬러, 색 파라미터, ToD, 엔진 색보정 노드만 남기고 새로 정리하여 구현. 비실사 파트를 Emissive 노드로 NPR 표현하고, 아웃라인은 Inverted Hull로 처리.

📷 뷰포트: VRM 캐릭터 툰 렌더링 결과 — Emissive NPR + Inverted Hull 아웃라인
📷 비교: 좌=VRM4U MToon 기본 렌더링 / 우=재구현 MToon(ToD, 색보정 적용)
Material Node Graph — VRM Emissive NPR
📷 머티리얼 에디터: VRM 툰 머티리얼 — MToon 파라미터 → Emissive 채널 NPR 출력 노드 전체

3. VRM PBR

VRM 캐릭터의 실사 파트(피부, 눈동자, 액세서리 등)에 사용하는 PBR 전용 머티리얼. BaseColor·Normal·Roughness 등 표준 PBR 텍스처를 사용하여, 하나의 VRM 캐릭터 안에서 툰 파트와 실사 파트가 자연스럽게 공존한다.

📷 VRM 캐릭터: 툰 파트만 렌더링 (비실사 부분)
📷 VRM 캐릭터: PBR 파트만 렌더링 (실사 부분)
📷 VRM 캐릭터: 툰+PBR 합성 최종 결과
Material Node Graph — VRM PBR
📷 머티리얼 에디터: VRM PBR 머티리얼 — BaseColor/Normal/Roughness 텍스처 연결 노드 전체

Matcap 리플렉션

NPR 환경에서는 PBR 리플렉션이 어울리지 않는다. Matcap 텍스처로 뷰 기반 반사를 구현하여, 머리카락 하이라이트나 피부 광택 등 NPR 스타일에 맞는 리플렉션을 적용했다.

📷 뷰포트: Matcap OFF — 머리카락/피부에 하이라이트 없음
📷 뷰포트: Matcap ON — 머리카락 하이라이트, 피부 광택 반사 보이는 동일 앵글
Material Node Graph
📷 머티리얼 에디터: ViewDirection → Transform → Matcap 텍스처 UV 변환 → 리플렉션 출력 노드 전체

마스크 기반 타투

아티스트가 마스크 텍스처를 그리면 캐릭터 피부 위에 타투가 렌더링되는 시스템. 마스크 채널별로 독립 타투를 제어할 수 있어 조합의 자유도가 높다.

기술 상세

NdotL (Normal dot Light)
표면 노멀 벡터와 라이트 방향의 내적. 값이 1이면 정면 조명, 0이면 경계, -1이면 뒷면. Cel Shading은 이 연속값을 단계별로 끊어서 2~3단계 밴드로 만든다.
Inverted Hull Outline
메시를 복제 → 노멀 방향으로 확장 → 뒷면만 렌더링 → 외곽선처럼 보임. 3D 공간에서 동작하여 카메라 거리에 따라 자연스러운 두께 변화.
