NPR Shader System

메타휴먼에서 애니메 스타일로 전환 — PBR 배경 위에 NPR 캐릭터를 분리 렌더링, 인하우스와 VRM 양쪽에 4종 룩 적용
Shader Development, Look Development, Material System 2023~2024 회사 프로젝트
Unreal Engine 5 Material Editor Post Process Stencil Buffer Custom Depth SDF Matcap

문제

프로젝트 초기 메타휴먼 기반 실사 렌더링에서 애니메 스타일로 전환이 결정됐다. PBR 배경은 유지하면서 캐릭터만 NPR로 렌더링해야 했고, 인하우스 캐릭터와 외부 VRM은 머티리얼 구조가 달라서 같은 방식으로는 양쪽 룩을 맞출 수 없었다.

해결

Cel Shading 기반 NPR 프로토타입을 만든 뒤, 시스템 확장과 유지보수를 주도했다.

인하우스는 Custom Depth + Post Process, VRM은 Emissive + Inverted Hull로 나눠 4종 NPR 룩을 만들었다. 프로토타입부터 유지된 핵심 노드는 SDF 얼굴 그림자와 Dot Product 기반 투톤 셰이딩이다.

베이스 머티리얼 4종

인하우스와 VRM은 머티리얼 구조가 다르고, VRM 안에서도 실사/비실사 파트가 섞여 있다. 그래서 네 종류의 베이스 머티리얼을 따로 만들었다.

1. 인하우스 툰

인하우스 캐릭터 전용 툰 머티리얼. Custom Depth 버퍼로 캐릭터를 분리하고, Post Process에서 NPR 스타일로 렌더링하는 구조 위에서 SDF 얼굴 그림자, Matcap 리플렉션 등 기능을 개발하고 유지보수를 담당했다.

Global Shadow OFF / ON 토글 데모

SDF 얼굴 그림자

일반 NdotL로는 코, 눈두덩이 주변에 비직관적인 그림자가 생긴다. SDF 텍스처에 미리 구운 거리 정보로 그림자 경계를 제어해서, 라이팅 각도가 바뀌어도 얼굴 그림자가 자연스럽게 떨어진다.

SDF 얼굴 그림자 라이팅 데모

SDF Shadow Mask 텍스처

Matcap 리플렉션

NPR 환경에서는 PBR 리플렉션이 어울리지 않는다. Matcap 텍스처로 뷰 기반 반사를 구현하여, 머리카락 하이라이트나 피부 광택 등 NPR 스타일에 맞는 리플렉션을 적용. 구현 후 페이크 림라이트 표현도 가능하다는 걸 발견해, 그 가능성을 탐구하기 위해 Matcap Painter를 만들었다.

Matcap 리플렉션 데모

2. VRM 툰

인하우스 툰 머티리얼을 기반으로 파라미터 인풋 이름을 MToon 규약에 맞춰 재구성. VRM 임포트 시 MToon 텍스처와 플래그가 자동으로 연결되도록 했다. 아웃라인은 인하우스와 동일하게 Post Process 기반.

3. VRM Zo Toon

기존 인하우스 머티리얼이 Custom Depth 버퍼에서 색을 잡는 방식이라 실사/비실사 혼합이 불가능했다. 이를 해결하기 위해 Emissive 채널에서 동작하는 툰 셰이딩을 새로 제작. 아웃라인은 Inverted Hull 방식. 기존 인하우스 캐릭터들과 톤을 맞추는 게 중요했다.

4. VRM PBR

VRM4U 플러그인의 PBR 머티리얼을 경량화하여 사용. 기본 SSS Profile 기준으로 피부, 눈동자, 액세서리 등 실사 파트를 렌더링. Zo Toon과 마찬가지로 기존 인하우스 캐릭터들과 톤을 맞추는 쪽으로 잡았다.

기술 상세

NdotL (Normal dot Light)
표면 노멀 벡터와 라이트 방향의 내적. 값이 1이면 정면 조명, 0이면 경계, -1이면 뒷면. Cel Shading은 이 연속값을 단계별로 끊어서 2~3단계 밴드로 만든다.
Inverted Hull Outline
메시를 복제 → 노멀 방향으로 확장 → 뒷면만 렌더링 → 외곽선처럼 보임. 3D 공간에서 동작하여 카메라 거리에 따라 자연스러운 두께 변화.
