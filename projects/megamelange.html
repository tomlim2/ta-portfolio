MegaMelange MCP

자연어로 야외 배경을 세팅하고, AI 이미지 생성으로 원하는 분위기를 뽑아내는 씬 구성 파이프라인
System Architecture, C++ Plugin, UE Integration 2025.8 ~ 2026.2 (6.5개월, 234 커밋) 회사 프로젝트 (POC)
Next.js 15.4 FastAPI C++ (UE Plugin) Anthropic SDK Google GenAI TCP Socket

문제

야외 씬을 구성하려면 지형(Cesium) 위치 세팅, 날씨/시간대(UDS/UDW) 조정, 라이팅, 카메라 배치를 모두 수동으로 해야 하는데, 이 과정을 비개발 직군도 빠르게 이터레이션할 수 있어야 했다.

해결

자연어 → AI 해석 → UE 엔진 제어, 3단계로 나눈 브릿지 구조. C++ 플러그인이 UE 에디터를 실시간으로 조작하고, AI가 자연어를 엔진 명령으로 바꿔준다.

"샌프란시스코로 가주세요"

"차가운 12시를 세팅해주세요"

핵심 — Cesium + UDS 연동

가장 공들인 부분. Cesium 지형 데이터에 Ultra Dynamic Sky를 붙여서, 자연어 한 줄로 위치 + 시간대 + 날씨 + 분위기가 세팅된 야외 배경을 만든다. 여기에 Nano Banana(i2i)를 걸면 원하는 컨셉이나 느낌으로 바꿀 수 있어서, 씬 구성부터 컨셉 아트까지 한 번에 뽑는다.

Roblox 연동 — 비즈니스 데모

Roblox API로 FBX를 받아 UE에 임포트 → 생성된 배경에 캐릭터 배치 → Nano Banana/Veo-3로 이미지/비디오 생성. CEO, 기획자, 팀장 대상 시연 콘텐츠.

기술 상세

UE C++ Plugin

8종 커맨드 핸들러 (6,300 LOC)
Actor, Blueprint, Lighting, Rendering, Camera, Cesium, UDS, UDW — 도메인별 핸들러가 TCP로 받은 JSON 명령을 UE API 호출로 바꿔서 실행.
TCP 소켓 프로토콜
JSON 기반 실시간 양방향 통신. TCP_NODELAY, keepalive, 65KB 버퍼, 청크 응답 지원. Python 백엔드 → UE 에디터 간 명령 전달 및 결과 수신.
